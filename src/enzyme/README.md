# TMI.jl Custom Enzyme Rules

This directory contains the source code for custom reverse-mode automatic differentiation rules for [Enzyme.jl](https://github.com/EnzymeAD/Enzyme.jl).

## Documentation

All documentation, including a full tutorial on the mathematics and implementation of these rules, can be found in the `tutorial` subdirectory.
