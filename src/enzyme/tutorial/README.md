# Enzyme Documentation for TMI.jl

This folder contains the documentation for using Enzyme.jl to perform Automatic Differentiation (AD) within the TMI.jl project.

The primary guide is `enzyme_tutorial.md`, which covers the theory and implementation of the custom AD rules used in TMI.

- **[A Guide to Automatic Differentiation in TMI with Enzyme.jl](./enzyme_tutorial.md)**: A comprehensive, lecture-style tutorial that explains the core concepts of reverse-mode AD and shows how they are applied to the key mathematical patterns in TMI.
