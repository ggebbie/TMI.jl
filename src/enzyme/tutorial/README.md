# TMI.jl Automatic Differentiation Tutorials

This directory contains the documentation for using Enzyme.jl to perform Automatic Differentiation (AD) within the TMI.jl project. The documentation is split into two parts, intended to be read in order.

### Part 1: The Theory of Automatic Differentiation

This document provides a textbook-style introduction to the theory of AD, explaining how it works from first principles. It is recommended for all users new to the concepts of reverse-mode AD and adjoints.

[**Read Part 1: The Theory of Automatic Differentiation](./analytical_ad.md)**

### Part 2: AD Implementation with Enzyme.jl in TMI

This document covers the practical application of AD in TMI.jl using the Enzyme.jl library. It explains how the concepts from Part 1 are implemented in code and details the specific custom rules used in TMI.

[**Read Part 2: AD Implementation with Enzyme.jl in TMI](./enzyme_implementation.md)**
